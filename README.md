## Описание проекта | Project Description
Эта игра является моей курсовой работой по дисциплине "Программирование" на первом курсе ВУЗа. Проект представляет собой компьютерную игру "Тетрис" со всеми основными механиками данной игры.
<br>
<br>
<br> 

This game is my first-year coursework for a Programming course at university. The project is a computer game called Tetris, complete with all the core mechanics.
<br>
<br>

Основной стэк технологий / Core technology stack:
<br>C#,
<br>Unity,
<br>DOTween,
<br>
<br>

## Основные возможности приложения | Main features of the application
Главное меню,
<br>Выбор из 3 игровых режимов,
<br>Сохранение и демонстрация лучших рекордов,
<br>Основные механики Тетриса,
<br>Отображение следующих 5 фигур,
<br>Рандомизация фигур с помощью тасование Фишера — Йетса,
<br>Возможность удерживать 1 фигуру,
<br>Пауза и меню паузы,

<br>Main menu,
<br>Choose from 3 game modes,
<br>Saving and displaying your best scores,
<br>Basic Tetris mechanics,
<br>Displaying the next 5 pieces,
<br>Piece randomization using the Fisher-Yates shuffle,
<br>The ability to hold one piece,
<br>Pause and pause menu,
<br>
<br>

## Управление | Controll:
A/D or Right/Left arrows - Горизонтальное движение | Horizontal Move 
<br>S or Down arrow - Быстрый спуск | Fast down
<br>R or Up arrow - Поворот | Rotare
<br>ESC - Пауза | Pause
<br>Space - Мгновенный спуск | Hard drop
<br>Shift - Удержание фигуры | Hold
